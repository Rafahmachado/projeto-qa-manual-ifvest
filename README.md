# Projeto de Testes Manuais – Plataforma Educacional IFVEST

Este repositório apresenta um projeto de testes manuais
baseado em atividades realizadas durante estágio acadêmico,
utilizando a plataforma educacional IFVEST como estudo de caso.

## Objetivo do Projeto
- Aplicar conceitos de qualidade de software
- Planejar e executar testes manuais
- Documentar casos de teste e defeitos
- Demonstrar experiência prática em QA

## Funcionalidades Testadas
- Cadastro de usuário
- Login
- Acesso aos simulados
- Navegação na plataforma

## Ferramentas Utilizadas
- Navegador web
- GitHub
- Markdown
